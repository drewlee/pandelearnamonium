# Pandelearnamonium

This project is a repository facilitating the learning and exploration of prominent
computer science and frontend development concepts.

It is set up as an NPM workspaces monorepo, consisting of two primary packages.

## Algorhythmics

[./packages/algorhythmics](./packages/algorhythmics)

- Data structures & algorithms.
- Coding challenge solutions.

## Cassess

[./packages/cassess](./packages/cassess)

- Modern CSS & React.
- Frontend Mentor projects.

## Development

Ensure code quality by running TypeScript, lint checks, and tests.

```sh
$ npm run test
```
